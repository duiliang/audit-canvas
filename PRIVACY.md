# Privacy

AuditCanvas is local-first.

- CLI output is written to `.auditcanvas/` in the current project.
- The Web demo uses local example data.
- Review state is stored in browser local storage in the current MVP.
- Remote AI providers are disabled by default.
- API keys must not be written to logs, exports, reports, examples, browser state, or Git history.

If you enable a remote Provider, you are responsible for reviewing what data is sent to that provider.
