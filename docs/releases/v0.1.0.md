# AuditCanvas v0.1.0

AuditCanvas 首个公开版本提供本地优先、感知 Git、保留完整证据的软件制品审计工作流。

## 主要内容

- 提供版本化 JSON Schema、稳定标识和确定性本地审计核心。
- 支持 Markdown、TXT、JSON 和源代码制品解析。
- 内置精确重复、规范化重复、近似重复和覆盖率检查。
- 提供 `scan`、`export`、`verify-coverage`、`doctor` 和 `serve` CLI 命令。
- 提供 React/Vite 审计工作台，包含制品导航、源码查看、问题审阅、证据对比、追踪矩阵、Git 差异和导出能力。
- 提供 Mock、OpenAI-compatible 和 Ollama 可选分析器适配器；远程分析默认关闭。
- 提供可独立安装的 Codex 插件，以及审计、基线比较和问题影响复核技能。
- 提供 CI、覆盖率门禁、Playwright 端到端测试、GitHub Pages 演示和发布校验。

## 安全与隐私

- 审计默认在本地运行。
- 只有用户显式选择远程分析器时，内容才会发送给配置的服务。
- 导出结果保留原始证据、路径和行号，不以省略号替换证据。
